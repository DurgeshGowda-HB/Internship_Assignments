# My Task Manager

A simple Task Manager app built using React.

---

## Features

- Add a task
- Delete a task
- Mark task as completed
- Shows "No tasks available" when list is empty
- Task counter

---

## How to Run

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

---

## Components Used

| Component | Description |
|-----------|-------------|
| `App.js` | Main component, stores all tasks |
| `TaskInput.js` | Input field and add button |
| `TaskList.js` | Shows all tasks |
| `TaskItem.js` | Single task with checkbox and delete |
